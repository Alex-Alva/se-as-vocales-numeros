# Señas IA - Reconocimiento de Lenguaje de Señas en Tiempo Real

Este es un proyecto de visión artificial interactivo desarrollado 100% en el frontend utilizando Inteligencia Artificial local. Permite el entrenamiento y evaluación del lenguaje de señas (vocales y números) directamente desde el navegador de forma privada y segura.

## Stack Tecnológico
*   **Framework:** React + Vite
*   **Estilos:** Tailwind CSS + PostCSS
*   **Estilos Visuales:** Soporte nativo para Modo Claro  y Modo Oscuro 
*   **Persistencia:** LocalStorage (para guardar el historial de muestras y configuraciones locales)


## Instalación y Configuración Local

Para ejecutar este proyecto en tu computadora, asegúrate de tener instalado [Node.js](https://nodejs.org/) y sigue estos pasos:

1. **Descargar y extraer** el archivo ZIP en tu computadora.
2. **Abrir una terminal** (consola) dentro de la carpeta raíz del proyecto.
3. **Instalar las dependencias** ejecutando el siguiente comando:
   npm install


## Iniciar el servidor de desarrollo local:

* npm run dev
